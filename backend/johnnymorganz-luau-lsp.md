## 📝 Summary

JohnnyMorganz/luau-lsp is an open‑source Language Server Protocol (LSP) implementation that provides full‑featured IDE support for the Luau scripting language, including diagnostics, autocomplete, hover information, and go‑to definition. Designed for backend integration, it enables editors and development tools to offer real‑time code analysis and refactoring capabilities for Luau projects. The server is actively maintained and can be easily incorporated into any workflow that supports the LSP standard.
