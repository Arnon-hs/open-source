## 📝 Summary

samanhappy/mcphub is an open‑source backend hub that centralizes the management of multiple Minecraft Proxy (MCP) servers and APIs, exposing each as its own endpoint. It provides dynamic orchestration and flexible routing strategies, allowing developers to configure, scale, and route traffic between MCP instances with minimal effort. The project streamlines deployment and maintenance of complex MCP architectures while keeping the setup modular and extensible.
