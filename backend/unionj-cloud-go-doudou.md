## 📝 Summary

Here's a brief summary for the open-source project "unionj-cloud/go-doudou":

go-doudou is a lightweight microservice framework that supports both REST and gRPC services, leveraging OpenAPI 3.0 and Protobuf v3 specifications. This framework enables the development of scalable and maintainable microservices, as well as monolith service applications. It offers a flexible and efficient approach to building cloud-native services.
