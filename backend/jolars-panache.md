## 📝 Summary

jolars/panache is an open‑source development toolkit that provides a language server, code formatter, and linter specifically for Markdown, Quarto, and R Markdown documents. It integrates with editors and IDEs to offer real‑time diagnostics, consistent styling, and intelligent language features, helping authors write cleaner, error‑free content. The project is positioned as a backend DevTools library for enhancing the authoring workflow of scientific and technical documents.
