## 📝 Summary

ScalarDB is an open-source, universal HTAP (Hybrid Transactional/Analytical Processing) engine that enables fast and seamless data processing for both online transactions and analytical queries. It aims to provide a unified platform for various use cases, from real-time analytics to complex transactions, with high performance and scalability. ScalarDB is designed to be highly adaptable and compatible with various data storage systems.
