## 📝 Summary

The DMOJ/judge-server is an open-source project that serves as the judging backend for the DMOJ online judge. It facilitates the evaluation of programming contest submissions, providing a robust and scalable infrastructure for competitive programming communities. This project enables the efficient management and scoring of contests, making it a valuable resource for developers and organizers.
