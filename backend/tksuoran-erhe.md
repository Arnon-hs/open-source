## 📝 Summary

The erhe project is an open-source C++ library and editor for 3D graphics that abstracts graphics APIs, providing a unified interface for Vulkan, Metal, and OpenGL. This abstraction allows developers to write cross-platform code with minimal API-specific modifications. erhe aims to simplify the development of 3D graphics applications by providing a flexible and efficient graphics API abstraction.
