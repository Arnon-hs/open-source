## 📝 Summary

Veriflow is a declarative API testing CLI designed for both humans and automated CI/CD pipelines. This open-source project enables users to define API behaviors using a simple, human-readable syntax, making it easier to test and verify complex APIs. With Veriflow, developers can write more efficient and effective API tests, streamlining their development workflows.
