## 📝 Summary

**nve** is a lightweight CLI tool that lets you execute any command under a specified Node.js version without manually switching environments. By automatically downloading and managing the requested Node versions, it streamlines testing, building, or running scripts across multiple Node releases, making it ideal for backend development and DevOps workflows.
