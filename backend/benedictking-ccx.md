## 📝 Summary

BenedictKing/ccx is an open‑source backend service that acts as a unified proxy for multiple large‑language‑model APIs, including Claude, Codex, and Gemini. It provides a single, consistent interface for sending requests and handling responses, simplifying integration and allowing developers to switch between providers without changing application code. The project is designed for easy deployment and extensibility, making it a handy tool for building AI‑powered applications.
