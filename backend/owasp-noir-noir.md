## 📝 Summary

Here's a brief summary for the open-source project "owasp-noir/noir":

Noir is an open-source tool designed to help developers and security teams identify and expose potential vulnerabilities in their code by hunting every endpoint and mapping the attack surface. It achieves this by scanning code for hidden or shadow APIs, providing a comprehensive view of the application's attack surface. By using Noir, teams can proactively identify and address security weaknesses before they can be exploited.
