## 📝 Summary

Codebadger is an open‑source backend service that runs Joern, a powerful code‑property graph engine, as an MCP (Message Control Protocol) server for static code analysis. It provides a simple API for submitting source code, generating graph‑based representations, and querying security‑relevant patterns, enabling automated detection of vulnerabilities across large codebases. The project is geared toward developers and security teams looking to integrate Joern’s analysis capabilities into CI/CD pipelines or custom tooling.
