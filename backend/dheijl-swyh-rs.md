## 📝 Summary

The dheijl/swyh-rs project is an open-source implementation of Stream What You Hear (SWYH) in Rust, allowing users to stream their system audio to other applications. This project aims to provide a reliable and efficient backend solution for SWYH functionality. It is designed to be a drop-in replacement for existing SWYH implementations, written in Rust for better performance and maintainability.
