## 📝 Summary

Real‑Estate MCP is a backend service that lets users query Korean apartment price data through natural‑language questions powered by Claude. It fetches up‑to‑date information from the Ministry of Land, Infrastructure and Transport’s open API, parses the results, and returns concise answers about property values. The project is open‑source and ready to be integrated into any application that needs Korean real‑estate pricing insights.
