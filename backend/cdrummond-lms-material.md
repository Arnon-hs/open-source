## 📝 Summary

CDrummond/lms‑material is an open‑source skin that gives the Lyrion Music Server (LMS) a clean, modern Material Design look and feel. It replaces the default LMS interface with responsive layouts, customizable colors, and enhanced navigation while preserving full backend functionality. The project is maintained as a community‑driven backend extension for LMS users who want a polished, mobile‑friendly UI.
