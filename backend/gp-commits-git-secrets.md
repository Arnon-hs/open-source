## 📝 Summary

The GP-commits/Git-Secrets project is an open-source Python CLI tool designed to identify and detect accidentally committed sensitive information in GitHub repositories owned by a user or organization. It scans for common secrets such as API keys, tokens, passwords, and private keys, helping to maintain the security and integrity of codebases. This tool is essential for developers and organizations to prevent data breaches and ensure compliance with security standards.
