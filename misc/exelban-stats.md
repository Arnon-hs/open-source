## 📝 Summary

A lightweight macOS utility that places a real‑time system monitor directly in the menu bar, showing key metrics such as CPU usage, memory pressure, network activity, and disk I/O. It’s open‑source, highly configurable, and designed to run unobtrusively with minimal resource overhead. Suitable for users who want quick performance insights without leaving the menu bar.
