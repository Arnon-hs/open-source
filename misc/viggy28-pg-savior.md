## 📝 Summary

pg_savior is an open-source project that aims to prevent accidental data modifications in PostgreSQL databases. It achieves this by blocking DELETE and UPDATE operations unless explicitly permitted by the database administrator. This project helps maintain data integrity and prevents unintended changes to critical data.
