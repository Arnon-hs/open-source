## 📝 Summary

OO is an open-source project that automates the process of shutting up Windows' Out of Memory (OOM) and other system errors, aiming to provide a more stable and seamless user experience. By silencing these errors, OO helps prevent system crashes and freezes caused by OOM conditions. This tool is designed to be user-friendly and can be easily integrated into Windows systems.
