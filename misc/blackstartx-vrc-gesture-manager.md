## 📝 Summary

BlackStartx/VRC‑Gesture‑Manager is an open‑source Unity extension that lets VRChat creators preview and edit their avatar animations in‑engine, eliminating the need for external testing. It provides a streamlined interface for managing gesture states, syncing animation clips, and visualizing transitions directly within the Unity editor, speeding up the avatar development workflow.
