## 📝 Summary

JohnnyMorganz/StyLua is an open‑source code formatter for the Lua programming language. It automatically reformats Lua scripts according to a consistent style guide, improving readability and reducing manual formatting effort. The tool is lightweight, highly configurable, and can be integrated into editors, CI pipelines, or run from the command line.
