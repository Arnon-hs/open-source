## 📝 Summary

Rex is an open‑source script execution engine that runs user‑provided code inside a hardened sandbox while enforcing fine‑grained access controls defined with AWS Cedar policies. By integrating Cedar’s declarative policy language, Rex lets developers specify exactly which resources a script may read, write, or invoke, ensuring secure and auditable execution of untrusted code. It’s a lightweight, language‑agnostic solution for safely automating tasks in environments where security and compliance are paramount.
