## 📝 Summary

Friend Maker is a cross‑platform (macOS/Windows) toolkit that converts any image into a pixel‑grid layout and generates the corresponding ESP32 controller script. The output can be used to automate drawing operations in Nintendo Switch’s Tomodachi Life and similar applications, streamlining the creation of custom pixel art directly on the console. It’s an open‑source utility aimed at hobbyists and developers interested in hardware‑assisted game interactions.
