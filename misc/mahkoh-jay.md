## 📝 Summary

mahkoh/jay is an open‑source Wayland compositor written to provide a lightweight, modular foundation for building custom desktop environments. It implements the core Wayland protocols while offering extensibility for advanced features such as input handling, output management, and compositor extensions. The project is positioned as a flexible, community‑driven alternative for developers and enthusiasts exploring Wayland graphics stacks.
