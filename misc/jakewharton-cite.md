## 📝 Summary

This project, cite, is an open-source Kotlin compiler plugin that allows developers to embed information about the file being compiled into their code. The tool is intended to provide metadata about the source code, such as the file name, path, and date of modification. By leveraging this plugin, developers can easily incorporate source information into their Kotlin applications.
