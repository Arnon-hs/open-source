## 📝 Summary

typst‑community/utpm is an open‑source package manager designed specifically for Typst, enabling users to install, update, and manage both local and remote Typst packages with a simple command‑line interface. It streamlines workflow by handling dependency resolution, version control, and package discovery, making it easy to share and reuse Typst resources across projects.
