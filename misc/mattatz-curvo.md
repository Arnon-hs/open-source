## 📝 Summary

The curvo library is an open-source, Rust-based project that provides a comprehensive framework for modeling and manipulating non-uniform rational B-spline (NURBS) curves and surfaces. This library offers a robust set of tools for creating, editing, and visualizing complex geometric shapes, making it a valuable resource for developers working in computer-aided design, computer-aided manufacturing, and related fields. By leveraging curvo, users can unlock the full potential of NURBS modeling in their applications.
