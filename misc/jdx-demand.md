## 📝 Summary

The jdx/demand project is an open-source library for Rust that provides a prompt library, enabling developers to easily create and manage interactive interfaces. This library allows users to define and execute tasks, or "demands," in a flexible and modular way. By utilizing this library, developers can simplify the process of building prompts and automating tasks within their Rust applications.
