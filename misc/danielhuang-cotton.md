## 📝 Summary

**cotton** is an open‑source package manager designed for JavaScript projects, offering a dramatically faster alternative to the traditional `rm -rf node_modules` cleanup workflow. By streamlining dependency installation and removal, `cotton install` reduces setup time and improves developer productivity. It’s a lightweight, miscellaneous‑category tool that can be integrated into any JavaScript development environment.
