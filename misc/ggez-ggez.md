## 📝 Summary

ggez is an open‑source Rust library that streamlines the development of 2D games by providing a simple, high‑level API for graphics, audio, input handling, and game loops. Designed with ergonomics and performance in mind, it abstracts away boilerplate code while still giving developers direct access to low‑level functionality when needed. The project aims to make creating "good games easily" accessible to both beginners and experienced Rust programmers.
