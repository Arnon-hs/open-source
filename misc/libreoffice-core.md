## 📝 Summary

LibreOffice /core is the read‑only mirror of the LibreOffice core source code, serving as a reference point for developers who contribute via LibreOffice’s Gerrit code‑review system rather than GitHub pull requests. The repository contains the full core implementation but advises users to obtain official builds from the LibreOffice bundles site and to submit changes through Gerrit at https://gerrit.libreoffice.org/. It is categorized as a miscellaneous open‑source project.
