## 📝 Summary

Java Native Access (JNA) is an open‑source library that enables Java programs to call native code (C/C++ libraries) directly without writing JNI glue code. By providing a high‑level API and automatic type mapping, JNA simplifies integration with system libraries and third‑party native APIs while remaining pure Java. It is widely used for tasks such as accessing OS functionality, hardware interfaces, and performance‑critical native routines.
