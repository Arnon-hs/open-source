## 📝 Summary

The paholg/subenum project is an open-source Rust macro that simplifies the process of creating enums with a subset of values from a parent enum. This macro streamlines the declaration of nested enums, reducing boilerplate code and enhancing code readability. It aims to make working with complex enum hierarchies more efficient and maintainable.
