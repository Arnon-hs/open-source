## 📝 Summary

colbymchenry/codegraph is an open‑source, pre‑indexed knowledge graph that stores code‑related information for use with Claude Code, enabling rapid, token‑efficient queries without external API calls. By keeping the entire graph locally, it eliminates network latency and privacy concerns while dramatically reducing the number of tokens required for code‑understanding tasks. The project is positioned as a lightweight, plug‑and‑play solution for developers who need fast, offline access to a rich code knowledge base.
