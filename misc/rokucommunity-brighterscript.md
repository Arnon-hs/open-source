## 📝 Summary

BrighterScript is an open-source superset of Roku's BrightScript language, designed to extend its functionality while maintaining backwards compatibility. This project allows developers to leverage the full capabilities of BrightScript, with additional features and improvements for enhanced productivity and efficiency in Roku application development. By enhancing the BrightScript language, BrighterScript aims to simplify the development process for Roku users.
