## 📝 Summary

The **shlima/keneetic-antifilter** repository provides a set of custom routing rules for Keenetic routers, enabling users to filter or block traffic from platforms such as YouTube, Instagram, and ChatGPT. By installing these rules, administrators can easily manage and restrict access to these services at the network level without needing additional hardware or complex configurations.
