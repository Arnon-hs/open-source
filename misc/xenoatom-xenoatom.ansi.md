## 📝 Summary

XenoAtom is an open-source, allocation-friendly .NET library that enables efficient generation, parsing, and manipulation of ANSI/VT escape sequences. This library provides a comprehensive set of tools, including a writer, markup, tokenizer, and ANSI-aware text utilities, making it a versatile solution for working with terminal output and text formatting. It is designed to be lightweight and memory-efficient, making it suitable for a wide range of applications.
