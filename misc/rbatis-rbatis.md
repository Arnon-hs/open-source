## 📝 Summary

rbatis/rbatis is an open-source, compile-time async dynamic SQL ORM (Object-Relational Mapping) framework. It enables developers to build high-performance, asynchronous database applications by generating efficient SQL queries at compile-time. This allows for improved performance, reduced overhead, and enhanced scalability in database-driven applications.
