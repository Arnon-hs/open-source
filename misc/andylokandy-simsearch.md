## 📝 Summary

The simsearch project is a simple and lightweight, in-memory fuzzy search engine that efficiently searches for similar strings. It provides a basic yet effective solution for finding matches in a dataset, making it a useful tool for various applications. This open-source project is particularly ideal for scenarios where fast and memory-friendly search functionality is required.
