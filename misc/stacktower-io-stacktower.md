## 📝 Summary

Stacktower is an open-source project that allows users to visualize package dependencies in the form of XKCD-style tower diagrams. The tool currently supports a range of programming languages, including Python, Rust, JavaScript, Ruby, PHP, Java, and Go, enabling developers to easily illustrate complex dependency relationships. By providing a simple and intuitive visualization, Stacktower helps developers better understand and manage their project's dependencies.
