## 📝 Summary

The free5gc project is an open-source implementation of a 5G core network, compliant with the 3GPP Release 15 (R15) standards. This project provides a free and customizable alternative to commercial 5G core solutions, enabling researchers, developers, and organizations to build and test 5G networks. It aims to accelerate the development and adoption of 5G technologies.
