## 📝 Summary

tomboy-ng is an open‑source, cross‑platform rewrite of the classic Tomboy note‑taking application, built with modern libraries and a clean, extensible architecture. It offers fast, markdown‑compatible plain‑text notes, full‑text search, tagging, and synchronization via standard file formats, while preserving the simplicity and low‑resource footprint of the original. The project aims to provide a reliable, actively maintained alternative for users who need a lightweight yet powerful personal wiki.
