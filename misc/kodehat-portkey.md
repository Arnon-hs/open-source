## 📝 Summary

kodehat/portkey is an open-source, simple web portal that serves as a customizable startup page. It allows users to display a collection of links and URLs, as well as add custom pages, all managed through a single configuration file. This lightweight portal provides a flexible and easy-to-use solution for creating personalized web gateways.
