## 📝 Summary

Pingcap/docs-cn is an open-source project providing comprehensive Chinese documentation for TiDB, TiKV, and PD, making it easier for users and developers in the China region to learn and implement these distributed database systems. This project aims to bridge the language gap by offering accurate and up-to-date documentation in Chinese. It facilitates the adoption and growth of TiDB and its ecosystem in the Chinese-speaking community.
