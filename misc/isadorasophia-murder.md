## 📝 Summary

Murder is an open-source, pixel art game engine that utilizes the Entity-Component-System (ECS) architecture. This engine provides a flexible and efficient framework for developers to create 2D games with a pixel art aesthetic. By leveraging ECS, Murder streamlines game development and enables the creation of complex game systems in a scalable and maintainable way.
