## 📝 Summary

horriblename/hyprgrass is an open‑source plugin that extends the Hyprland Wayland compositor with native touch‑screen gesture support. It detects common multi‑finger swipes, pinches and taps, translating them into configurable Hyprland actions such as workspace switching, window resizing, or custom command execution. The project is lightweight, written in C++ and licensed under the MIT license, making it easy to integrate and adapt for any Hyprland setup.
