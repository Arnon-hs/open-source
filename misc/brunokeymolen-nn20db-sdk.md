## 📝 Summary

This open-source project enables efficient vector search on the ESP32P4 microcontroller, a device with limited available RAM. By implementing the HNSW (Hierarchical Navigable Small World) algorithm, the project allows for fast similarity searches in large datasets that would otherwise not fit within the device's memory constraints. This innovation has the potential to accelerate applications in AI and machine learning, such as image or speech recognition, on resource-constrained edge devices.
