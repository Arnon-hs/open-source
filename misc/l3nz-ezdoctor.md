## 📝 Summary

Ezdoctor is a lightweight, single‑file tool that bundles Asciidoctor, diagram generation, and syntax‑highlighting into a self‑contained script, allowing users to convert AsciiDoc sources to richly formatted HTML or PDF without complex dependencies. Ideal for quick documentation workflows, it parses AsciiDoc markup, renders PlantUML/Graphviz diagrams, and applies Pygments‑style code highlighting, all from one executable file.
