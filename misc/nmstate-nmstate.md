## 📝 Summary

Nmstate is an open‑source library (with a companion CLI) that enables declarative configuration of a host’s networking stack—allowing users to define desired network states in YAML/JSON and apply them atomically. It abstracts low‑level Linux networking tools (NetworkManager, ip, ethtool, etc.) to provide a consistent, programmatic interface for managing interfaces, bonds, bridges, VLANs, and other network resources across diverse environments.
