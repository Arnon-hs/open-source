## 📝 Summary

CnPack/VCL (cnpack/cnvcl) is an open‑source collection of Delphi VCL components and utilities that extend the standard Visual Component Library with a variety of visual controls, editors, and helper classes. Designed for ease of integration, the package provides ready‑to‑use components such as enhanced grids, custom dialogs, and data‑binding tools, all under a permissive license. It aims to accelerate Delphi application development by offering reusable, well‑documented UI building blocks.
