## 📝 Summary

mrkite/minutor is an open‑source utility that generates detailed, real‑time maps of Minecraft worlds, allowing players to explore terrain, locate structures, and visualize biomes from a top‑down perspective. It supports multiple Minecraft versions, offers customizable rendering options, and can export maps as images or interactive web pages for easy sharing. The project is lightweight, cross‑platform, and licensed under an open‑source agreement, making it a handy tool for both casual gamers and server administrators.
