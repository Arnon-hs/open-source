## 📝 Summary

LavinMQ is an open-source message broker written in the Crystal programming language. It aims to provide a lightweight and efficient alternative for message queuing and communication between applications. By leveraging Crystal, LavinMQ is designed to offer high performance and scalability while maintaining ease of use.
