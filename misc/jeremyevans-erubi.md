## 📝 Summary

Erubi is a lightweight, open-source ERB (Embedded RuBY) implementation that allows users to embed Ruby code within templates for dynamic content generation. This project is designed to be a compact and efficient alternative to larger ERB libraries. By utilizing Erubi, developers can efficiently render dynamic web pages with minimal overhead.
