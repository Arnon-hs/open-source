## 📝 Summary

umijs/dumi is an open‑source static site generator tailored for building component libraries and documentation sites. It provides a zero‑config development environment with built‑in Markdown support, live preview, and automatic API extraction, enabling developers to quickly create, test, and publish component demos and docs. The tool integrates seamlessly with the Umi ecosystem while remaining framework‑agnostic for broader usage.
