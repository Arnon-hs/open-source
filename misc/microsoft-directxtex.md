## 📝 Summary

DirectXTex is an open‑source C++ library that provides a comprehensive set of tools for loading, saving, converting, and manipulating texture data used in DirectX applications. It supports a wide range of image formats (including DDS, TGA, BMP, PNG, JPEG, and HDR) and offers utilities for generating mipmaps, compressing textures, and performing format conversions across various DXGI formats. The library is lightweight, cross‑platform (Windows and UWP), and is widely used in game development and graphics tooling.
