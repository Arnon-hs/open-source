## 📝 Summary

Git LFS is an open-source extension for Git that allows version control of large files, making it easier to manage and collaborate on projects with large data sets. It provides a way to store and manage binary files, such as images, videos, and audio files, alongside your code. By separating large files from your repository, Git LFS helps to improve performance and reduce storage costs.
