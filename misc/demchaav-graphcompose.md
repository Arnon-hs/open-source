## 📝 Summary

GraphCompose is an open-source, declarative PDF layout engine written in Java and released under the permissive MIT license. This project allows developers to create complex, custom PDF layouts in a straightforward and flexible manner. By using a declarative approach, GraphCompose streamlines the process of designing and generating PDFs with precise control over layout and formatting.
