## 📝 Summary

Togatoga / karukan is an open‑source Japanese input method for Linux that combines a neural‑network‑based Kana‑Kanji conversion engine with the modern Fcitx5 framework. It provides fast, context‑aware predictions and seamless integration into desktop environments, offering a lightweight yet powerful alternative to traditional Japanese IMEs.
