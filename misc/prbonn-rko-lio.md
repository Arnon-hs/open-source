## 📝 Summary

PRBonn / rko_lio is an open‑source implementation of a robust LiDAR‑Inertial Odometry system that avoids the need for sensor‑specific calibration or modeling. By tightly coupling raw LiDAR scans with inertial measurements through a generic optimization framework, it delivers accurate, drift‑reduced pose estimates across a wide range of hardware setups. The project provides ready‑to‑run code, documentation, and evaluation tools for researchers and developers working on autonomous navigation.
