## 📝 Summary

The halley game engine is a lightweight, open-source project written in modern C++. It aims to provide a simple, efficient framework for game development, catering to the needs of indie game developers and hobbyists. Halley is designed to be easy to use, flexible, and highly customizable, allowing developers to create a wide range of 2D and 3D game experiences.
