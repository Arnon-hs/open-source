## 📝 Summary

This open-source project, nodeca/probe-image-size, provides a solution to determine the size of various image types without fully downloading them. It supports a wide range of image formats, including JPG, GIF, PNG, WebP, BMP, TIFF, SVG, PSD, and ICO. This allows for efficient and lightweight image size checks in web development and other applications.
