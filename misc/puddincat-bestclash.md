## 📝 Summary

PuddinCat/BestClash is an open‑source tool that automatically gathers free proxy nodes from the web, tests their speed, and generates a Clash (or Clash Meta/Mihomo) subscription that refreshes every 30 minutes. It provides users with up‑to‑date, ready‑to‑use VPN/proxy configurations for scientific internet access, eliminating the need for manual node hunting.
