## 📝 Summary

Sql2md is a lightweight, zero‑dependency utility that converts raw SQL queries and schema definitions into cleanly formatted Markdown tables and code blocks. Ideal for documentation pipelines, it parses CREATE statements, SELECT results, and other SQL fragments without requiring external libraries or runtime installations. The tool is open‑source, easy to integrate into any workflow, and works across platforms straight from the command line.
