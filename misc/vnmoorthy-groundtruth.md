## 📝 Summary

Groundtruth is an open-source project that aims to stop the AI model Claude Code from displaying the "done" message prematurely. This project uses a hook to block or delay the completion message, allowing users to interact with the AI model for a longer period. Developed as a community-driven solution, Groundtruth is a useful tool for those who want to get the most out of their interactions with Claude Code.
