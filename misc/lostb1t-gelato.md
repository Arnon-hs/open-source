## 📝 Summary

The Gelato project is an open-source plugin that integrates Jellyfin with Stremio, allowing users to access and stream Jellyfin content directly within the Stremio interface. This integration enables seamless and feature-rich media management across both platforms. By leveraging this plugin, users can enjoy a unified streaming experience with access to Jellyfin's rich library of content within Stremio.
