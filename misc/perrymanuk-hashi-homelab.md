## 📝 Summary

This open-source project, "hashi-homelab," is a lightweight and compact homelab setup built using Nomad and Consul, two tools from HashiCorp. Designed for simplicity and ease of use, it provides a robust foundation for managing and orchestrating various services and applications in a home or small-scale environment. By leveraging the power of Nomad and Consul, users can create a highly available and scalable homelab infrastructure.
