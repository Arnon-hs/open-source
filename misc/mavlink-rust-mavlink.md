## 📝 Summary

MAVLink is a lightweight communication protocol widely used in UAV and robotics applications. The **rust-mavlink** project provides a pure‑Rust implementation of this protocol, offering tools for encoding, decoding, and handling MAVLink messages with idiomatic, type‑safe Rust APIs. It enables developers to integrate MAVLink support into Rust‑based ground stations, autopilots, and other embedded or desktop systems.
