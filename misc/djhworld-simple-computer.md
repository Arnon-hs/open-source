## 📝 Summary

A lightweight, open‑source implementation of the Scott CPU described in J. Clark Scott’s *But How Do It Know?*. The project provides a simple, extensible simulator that models the CPU’s instruction set, registers, and control logic, making it easy for students and hobbyists to explore fundamental computer architecture concepts.
