## 📝 Summary

The Vitals project is an open-source tool that provides a comprehensive overview of a computer's vital statistics. It offers real-time monitoring of temperature, voltage, fan speed, memory usage, and CPU load, allowing users to track their system's performance and health. This project is designed to be easy to use and provides valuable insights for system administrators and enthusiasts alike.
