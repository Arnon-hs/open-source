## 📝 Summary

PiSugar Power Manager is a Rust‑based utility for managing and monitoring PiSugar battery modules on Raspberry Pi devices. It provides a lightweight, open‑source command‑line interface and API to query battery status, control charging, and handle power‑related events. Designed for reliability and extensibility, the project aims to simplify power management for embedded and hobbyist Pi setups.
