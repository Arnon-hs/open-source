## 📝 Summary

Fc is an open‑source utility that compresses streams of floating‑point numbers without any loss of precision, using specialized encoding techniques to achieve high compression ratios while preserving the original data exactly. It is designed for fast, lossless compression of scientific, financial, or sensor data where maintaining exact binary fidelity is essential, and can be integrated into pipelines that handle large binary float datasets.
