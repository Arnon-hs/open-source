## 📝 Summary

Kiorg is a lightweight, terminal‑oriented file manager that blends Vim‑style navigation with modern conveniences such as zoxide‑like “teleport” shortcuts for rapid directory jumps and inline previews for files and directories. Designed for power users who prefer keyboard‑driven workflows, it offers a configurable, scriptable interface while remaining simple enough for everyday file‑browsing tasks. The project is open‑source and positioned as a versatile utility for a wide range of Unix‑like environments.
