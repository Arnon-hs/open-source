## 📝 Summary

OpenDream is an open‑source interpreter and runtime for games written in the DM (Dream Maker) language, the scripting language used by BYOND. It enables developers to run, test, and host DM‑based games on modern platforms without requiring the original BYOND client, providing compatibility layers and tooling for seamless execution. The project is community‑driven and aims to preserve and extend the life of classic DM games.
