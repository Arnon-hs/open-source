## 📝 Summary

linebender/resvg is a high‑performance, pure‑Rust library for parsing and rasterizing SVG (Scalable Vector Graphics) files. It provides a standards‑compliant rendering engine that can be used as a standalone command‑line tool or integrated into other applications, delivering fast, accurate SVG output across platforms.
