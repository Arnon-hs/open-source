## 📝 Summary

modular/mojo-gpu-puzzles is an open‑source learning platform that teaches GPU programming using the Mojo language through a series of interactive puzzles. Each challenge guides users to write and optimize Mojo kernels, providing instant feedback and visualizations of GPU execution. The project serves as a hands‑on tutorial for developers looking to master Mojo’s GPU capabilities in a fun, puzzle‑based format.
