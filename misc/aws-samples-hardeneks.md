## 📝 Summary

The hardeneks project is an open-source tool designed to ensure that Amazon Elastic Kubernetes Service (EKS) clusters adhere to the recommended best practices. It conducts a series of checks to identify potential security and configuration issues, providing actionable feedback to improve the overall security and reliability of EKS clusters. By utilizing hardeneks, users can proactively harden their EKS clusters and reduce the risk of security vulnerabilities.
