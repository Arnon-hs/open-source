## 📝 Summary

The dividebysandwich/shareboxx project is an open-source, anonymous file-sharing system that operates locally over WiFi, providing a secure and private platform for users to exchange files without internet connectivity. Written entirely in Rust, this system offers a robust and efficient alternative to existing solutions like Piratebox and Librarybox. By leveraging disconnected local sharing, users can maintain their privacy and security in a fully offline environment.
