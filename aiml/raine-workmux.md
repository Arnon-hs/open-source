## 📝 Summary

Raine / workmux is an open‑source tool that combines Git worktrees with tmux window management to enable developers to spin up multiple, isolated development environments instantly and switch between them without friction. By automating the creation of a worktree for each feature branch and attaching a dedicated tmux pane or window, it streamlines parallel development, testing, and code review workflows. The project is lightweight, script‑driven, and fits naturally into any Unix‑like workflow.
