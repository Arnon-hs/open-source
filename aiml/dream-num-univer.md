## 📝 Summary

Univer is an open‑source, full‑stack framework that lets developers build AI‑native spreadsheet applications for both web and server environments. It provides a unified platform where spreadsheets can be created, edited, and manipulated directly via natural‑language commands, combining frontend UI components with backend processing. Ideal for AI/ML‑driven data workflows, Univer streamlines the integration of conversational interfaces into traditional spreadsheet functionality.
