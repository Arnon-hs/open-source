## 📝 Summary

TartaricAcid’s **Touhou Little Maid** is an open‑source Minecraft Forge mod that introduces a fully animated maid NPC inspired by the Touhou Project. The maid can follow, assist, and interact with players, offering customizable behaviors, inventory management, and AI‑driven tasks such as gathering resources or defending the player’s base. The project’s source code is publicly available, inviting contributions and extensions from the community.
