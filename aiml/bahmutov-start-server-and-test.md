## 📝 Summary

**Summary:**  
`bahmutov/start-server-and-test` is a lightweight utility that launches a development server, waits until a specified URL becomes reachable, and then executes a test command (or any other script). After the test process finishes—whether it succeeds or fails—the tool automatically shuts down the server, streamlining the workflow for backend and dev‑tool environments.
