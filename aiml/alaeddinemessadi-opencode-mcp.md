## 📝 Summary

AlaeddineMessadi/opencode-mcp provides an MCP (Message Control Protocol) server that powers the OpenCode AI ecosystem, bundling 70 ready‑to‑use AI tools, 10 curated resources, and 5 prompt templates. By running `npx opencode-mcp`, developers can instantly connect any MCP‑compatible client—such as Claude Desktop, Claude Code, Cursor, Windsurf, or custom clients—to leverage these capabilities for backend, AI/ML, and developer tooling workflows.
