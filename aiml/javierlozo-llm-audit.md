## 📝 Summary

The llm-audit project is an open-source initiative that provides Semgrep rules for auditing large language models (LLMs) against the OWASP LLM Top 10, a list of common vulnerabilities and weaknesses in LLMs. This set of rules, written in TypeScript, aims to help developers identify and mitigate potential security risks in their LLMs. By using llm-audit, developers can ensure their LLMs are secure and compliant with best practices.
