## 📝 Summary

VoiceGoat is an open-source project that simulates a vulnerable voice agent designed to help developers and researchers practice and demonstrate Large Language Model (LLM) attacks. This project allows users to test and experiment with various attack vectors without posing a risk to real-world systems. By using VoiceGoat, users can hone their skills and stay ahead of emerging threats in the field of AI and machine learning.
