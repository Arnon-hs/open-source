## 📝 Summary

Domoticz is a free, open‑source home automation platform that runs on Linux, Windows and Raspberry Pi, providing a lightweight web interface for monitoring and controlling smart devices. It natively supports a wide range of protocols—including Z‑Wave, Zigbee, MQTT, and more than 150 compatible devices—allowing users to create custom automation rules and dashboards. Although primarily an IoT hub, its extensible architecture also enables integration with AI/ML services for advanced analytics and predictive control.
