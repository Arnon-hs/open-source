## 📝 Summary

mybigday/llama.rn provides a React Native wrapper for the llama.cpp library, enabling developers to run LLaMA‑based language models directly on iOS and Android devices. By exposing the core inference functionality through a simple JavaScript API, the project brings on‑device AI/ML capabilities to mobile front‑end applications without requiring a server backend. It’s an open‑source solution for integrating powerful LLM inference into cross‑platform mobile apps.
