## 📝 Summary

CodexIsland is an open‑source macOS utility that monitors and enforces AI usage limits directly from the notch area of your MacBook. By integrating with system APIs, it provides real‑time feedback and controls to help users track and restrict AI‑driven applications, promoting healthier digital habits. The project is licensed for community contributions and can be customized to set personalized usage thresholds.
