## 📝 Summary

Gerbil is an open-source desktop application that enables users to run Large Language Models locally, providing a seamless and private experience for text generation and processing tasks. This project offers a user-friendly interface for interacting with powerful language models, allowing for flexible and efficient use of AI capabilities. By running models locally, Gerbil ensures that sensitive data remains secure and under user control.
