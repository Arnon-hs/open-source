## 📝 Summary

UniClaudeProxy is an open‑source proxy that lets you run any LLM through Claude’s API format, translating Anthropic requests into calls for OpenAI, Gemini, DeepSeek, Ollama, and other providers. It supports full tool‑calling, streaming responses, ReAct‑style XML fallbacks, and hot‑reloading of configuration, making it a versatile bridge for both frontend and backend AI applications.
