## 📝 Summary

The easystats/performance project is an open-source initiative that provides a comprehensive collection of metrics for evaluating the quality and performance of machine learning models. This includes metrics such as R-squared (R2), Intraclass Correlation Coefficient (ICC), Leave-One-Out cross-validation (LOO), Akaike Information Criterion (AIC), and Bayes Factor (BF), among others. By offering a standardized framework for model evaluation, easystats/performance aims to facilitate more effective model selection and improvement in machine learning applications.
