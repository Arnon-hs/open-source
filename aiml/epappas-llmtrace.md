## 📝 Summary

Here is a brief summary of the epappas/llmtrace project:

LLMtrace is an open-source, zero-code proxy for OpenAI-compatible APIs that provides real-time security and observability features. It detects prompt injection attempts, scans for personally identifiable information (PII), and enables cost control, ensuring a secure and transparent experience with large language models. Built in Rust, LLMtrace offers a robust solution for developers and organizations to monitor and manage their LLM usage.
