## 📝 Summary

Gridwatch is an open-source, Tron-themed desktop application that provides a visual interface for managing GitHub Copilot CLI sessions. It enables users to browse their session history, track token usage, and explore activity across all their coding sessions in a convenient and interactive way. This tool is designed to simplify the management of GitHub Copilot CLI sessions and enhance productivity for developers.
