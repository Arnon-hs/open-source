## 📝 Summary

swyxio/swyxdotio is the source repository for the swyx.io blog, where each blog post is authored as a GitHub issue and automatically rendered as a static page on the site. The project showcases a lightweight, issue‑driven publishing workflow that integrates content creation, version control, and community interaction directly within GitHub. It serves as a practical example of using modern frontend tooling and CI to turn markdown issues into a searchable, comment‑enabled blog.
