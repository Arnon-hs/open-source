## 📝 Summary

Nezha is an open‑source code editor designed for the AI‑driven development era, enabling developers to run multiple Claude Code and Codex agents side‑by‑side on local projects. It streamlines collaboration between AI agents and human programmers, providing a unified interface to manage, invoke, and monitor various AI coding assistants across different repositories. The tool aims to boost productivity by integrating AI‑generated code suggestions, completions, and debugging directly into the development workflow.
