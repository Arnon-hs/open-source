## 📝 Summary

Minuet-ai.nvim is an open-source project that provides intelligent code completion as-you-type, leveraging popular large language models (LLMs) such as OpenAI and Gemini. This plugin offers seamless integration with Neovim, enabling developers to harness the power of AI-driven code suggestions. By supporting a wide range of LLMs, Minuet-ai.nvim enhances coding efficiency and accuracy.
