## 📝 Summary

MarCognity-AI is an open-source project that provides an epistemic layer for Large Language Models (LLMs), allowing users to run these models locally without relying on external APIs. This solution enables developers to maintain control and data security while utilizing the capabilities of LLMs. By running locally, users can enjoy faster performance and more seamless integration with their applications.
