## 📝 Summary

0xMassi/webclaw is a high‑performance, Rust‑based toolkit for extracting and structuring web content locally, designed to feed large language models. It offers a flexible command‑line interface, a REST API, and an MCP server for scraping, crawling, and converting pages into structured data without relying on external services. The project targets AI/ML pipelines, backend developers, and dev‑tool creators who need fast, offline web data extraction.
