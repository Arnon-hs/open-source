## 📝 Summary

NVIDIA‑NeMo Automodel is an open‑source library that enables native distributed training of large language and vision‑language models using PyTorch. It provides out‑of‑the‑box integration with Hugging Face Transformers, simplifying the scaling of LLM/VLM workloads across multiple GPUs or nodes. The project is geared toward AI/ML researchers and engineers who need efficient, production‑ready distributed training pipelines.
