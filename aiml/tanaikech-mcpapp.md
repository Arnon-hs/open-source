## 📝 Summary

tanaikech/MCPApp is an open‑source proof‑of‑concept that demonstrates how the Model Context Protocol (MCP) can be used for AI interactions via a Google Apps Script (GAS) backend. The repository provides a minimal working example, highlights the current absence of an official GAS SDK, and invites developers to explore and extend the implementation.
