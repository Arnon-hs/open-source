## 📝 Summary

The engram.ai project is an open-source, neuroscience-grounded memory framework for AI agents. It integrates cognitive mechanisms such as ACT-R activation, Ebbinghaus forgetting, and Hebbian learning to simulate human-like memory consolidation. Built on a Rust core with Python and TypeScript bindings, engram.ai provides a flexible and scalable platform for AI research and development.
