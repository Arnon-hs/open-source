## 📝 Summary

CodexBar is an open‑source browser extension that displays real‑time usage statistics for OpenAI Codex and Claude Code directly in the developer’s toolbar, eliminating the need to log into each service to monitor activity. It aggregates request counts, token usage, and cost estimates, giving AI‑assisted coding tools a transparent, at‑a‑glance overview for better resource management.
