## 📝 Summary

twpayne/go-jsonstruct is an open‑source utility that automatically generates Go struct definitions from one or more JSON or YAML inputs, intelligently merging differing field types and handling nested structures. It streamlines data‑model creation for Go developers, reducing manual coding and minimizing errors when working with complex or evolving JSON/YAML payloads. The tool is especially useful in AI/ML pipelines where schema generation from sample data is a common task.
