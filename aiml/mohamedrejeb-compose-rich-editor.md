## 📝 Summary

The MohamedRejeb/compose-rich-editor project is an open-source library that provides a customizable rich text editor for both Jetpack Compose and Compose Multiplatform applications. This library allows developers to create and edit text content in HTML and Markdown formats, offering flexibility and versatility in text editing capabilities. It is suitable for mobile and frontend applications, especially those requiring AI/ML integration.
