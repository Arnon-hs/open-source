## 📝 Summary

Yggdrasil is an open-source architecture enforcement tool designed to maintain code integrity with AI coding agents. It allows users to define rules in plain Markdown, which are then verified by a reviewer and fed back into the AI agent's loop for correction. This approach ensures that AI-generated code adheres to established standards and best practices.
