## 📝 Summary

The sapcc/keppel project is an open-source, regionally federated multi-tenant container image registry designed to provide a scalable and secure solution for managing container images across different regions. This project enables multiple tenants to share a single registry while maintaining regional control and compliance. It is particularly useful for organizations with global operations and diverse regulatory requirements.
