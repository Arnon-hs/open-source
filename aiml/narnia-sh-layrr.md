## 📝 Summary

Layrr (narnia‑sh/layrr) is an open‑source browser extension that lets developers interactively select page elements and issue commands straight to Claude Code, turning the AI into a live coding assistant for web projects. By bridging the DOM with Claude’s programming capabilities, Layrr streamlines debugging, prototyping, and UI tweaks directly from the browser, making AI‑driven development faster and more intuitive.
