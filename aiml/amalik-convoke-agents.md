## 📝 Summary

Convoke is an open-source project that extends the capabilities of BMAD Method AI agents through installable modules. It allows users to bring in new domain-specific agents (Teams) or add new skills to existing agents (Skills), enabling flexibility and customization. These modules can be installed independently or combined to create a tailored AI solution.
