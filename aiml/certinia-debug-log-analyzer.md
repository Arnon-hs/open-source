## 📝 Summary

certinia/debug-log-analyzer is an open‑source VS Code extension that parses Salesforce Apex debug logs and visualizes the data in real‑time. It delivers blazing‑fast analysis with interactive flame charts, detailed breakdowns of SOQL/DML operations, Apex performance insights, and AI‑enhanced recommendations, helping developers diagnose and optimize their Salesforce code more efficiently.
