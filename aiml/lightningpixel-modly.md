## 📝 Summary

LightningPixel’s **Modly** is a desktop application that creates 3D models directly from 2D images using locally‑run AI models, leveraging your GPU for fast inference without any cloud services. It provides an easy‑to‑use interface for converting photos into detailed meshes, making 3D asset generation accessible for creators, developers, and hobbyists while keeping data private and processing fully offline.
