## 📝 Summary

Here's a summary of the project:

Evidently is an open-source observability framework for machine learning (ML) and large language model (LLM) systems, enabling the evaluation, testing, and monitoring of AI-powered pipelines. It provides a comprehensive set of over 100 metrics to track the performance and behavior of these systems, from tabular data to generative AI models. With Evidently, data scientists and developers can gain insights into their AI systems and make data-driven decisions to improve their performance and reliability.
