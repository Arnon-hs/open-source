## 📝 Summary

LearnPrompt/ai-news-radar is an open‑source, 24‑hour AI and tech news aggregator that automatically collects, filters, and displays the latest headlines using GitHub Actions. It provides a live, responsive web UI for browsing the curated feed and includes a Scout Skill integration to query AI‑generated sources directly from the platform. The project is geared toward AI/ML enthusiasts and frontend developers looking for a continuously updated news radar.
