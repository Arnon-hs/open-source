## 📝 Summary

xintaofei/codeg is an open‑source tool that collects and visualizes AI coding agent sessions—from Claude Code, Codex, Gemini CLI, and others—into a single, searchable interface. It can be run as a desktop application, a self‑hosted server, or via Docker, making it easy to browse, manage, and replay AI‑generated code interactions across multiple platforms.
