## 📝 Summary

Here's a brief summary for the ClaudeWatch open-source project:

ClaudeWatch is a free, open-source menu bar tracker for Claude Code, providing real-time insights into live sessions and weekly usage, as well as a breakdown of all model types and Sonnet usage. This macOS application offers behavior analysis over the last 24 hours and automatically detects user plans (Claude Max, Pro, or Team). Built using SwiftUI and Python, it requires zero API tokens, making it a seamless tool for Claude users.
