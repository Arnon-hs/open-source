## 📝 Summary

The Claude-proxy project is an open-source proxy server that enables interaction with various AI models, including DeepSeek, Qwen, GLM, MiniMax, and Kimi, via the Anthropic Messages API. This proxy provides an interface for developers to easily access and utilize these models in their applications. By leveraging the Claude Agent SDK, the project streamlines communication between applications and the respective AI models.
