## 📝 Summary

tsParticles is an open‑source JavaScript library that lets developers generate highly customizable particle effects, confetti bursts, fireworks, and animated backgrounds for web pages. It provides ready‑to‑use components for a wide range of frameworks—including React, Vue (2.x & 3.x), Angular, Svelte, jQuery, Preact, Inferno, Solid, Riot, and standard Web Components—making it easy to integrate sophisticated visual animations into any frontend project.
