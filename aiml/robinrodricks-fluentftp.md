> An FTP and FTPS client for .NET & .NET Standard, optimized for speed. Provides extensive FTP commands, File uploads/downloads, SSL/TLS connections, Automatic directory listing parsing, File hashing/checksums, File permissions/CHMOD, FTP proxies, FXP support, UTF-8 support, Async/await support, Powershell support and more. Written entirely in C#.

## 📝 Summary

Here's a 2-3 sentence summary for the FluentFTP project:

FluentFTP is a high-performance, open-source FTP and FTPS client library for .NET and .NET Standard applications. It offers a comprehensive set of features, including SSL/TLS connections, automatic directory listing parsing, and file hashing, making it an ideal choice for developers working with file transfers. With its extensive FTP command support and async/await capabilities, FluentFTP streamlines file uploads, downloads, and directory management tasks.
