## 📝 Summary

Dasel is an open-source, multi-format data selector tool that allows users to effortlessly select, put, and delete data from various file formats including JSON, TOML, YAML, XML, INI, HCL, and CSV. Available as a CLI tool and as a Go module, Dasel simplifies data manipulation across different formats. This versatile tool is particularly useful for data scientists and developers working with AI, ML, and other data-intensive projects.
