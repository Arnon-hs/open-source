## 📝 Summary

Covertness / coap‑rs is an open‑source implementation of the Constrained Application Protocol (CoAP) written in safe, idiomatic Rust. The library provides a lightweight, asynchronous client and server API designed for resource‑constrained environments such as IoT devices, while leveraging Rust’s performance and memory‑safety guarantees. It aims to simplify the development of interoperable, low‑overhead network services that adhere to the CoAP standard.
