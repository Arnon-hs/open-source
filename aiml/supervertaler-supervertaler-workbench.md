## 📝 Summary

Supervertaler Workbench is an open‑source, AI‑enhanced computer‑assisted translation (CAT) platform that supports multiple large language models, translation memories, and glossary management, while offering a system‑wide “Sidekick” floating AI assistant for real‑time suggestions. It also includes advanced features such as the Superbench benchmarking suite, Superlookup concordance across TMs, glossaries and web resources, voice‑command control, an Okapi sidecar for file extraction, and seamless integration with popular CAT tools.
