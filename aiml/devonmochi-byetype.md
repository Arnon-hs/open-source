## 📝 Summary

ByeType is a Markdown‑driven AI voice input tool that leverages multimodal large models to process raw audio directly (using your own API key). By editing a Markdown file you can define custom vocabularies, transcription rules, and formatting strategies so that names, technical terms, filler words, and numeric formats are transcribed correctly in a single pass, minimizing post‑editing. The tool runs on macOS, Windows, and iOS.
