## 📝 Summary

The verbatim-rag project is an open-source solution that prevents hallucinations in AI-generated content by using a RAG (Retrieval-Augmented Generator) system. This system extracts exact citations from source documents to ensure generated content is grounded in verifiable evidence. By doing so, verbatim-rag promotes transparency and accountability in AI-generated text.
