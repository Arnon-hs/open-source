## 📝 Summary

A lightweight graphical interface for querying PostgreSQL databases, built on the GPUI framework. It lets users compose, run, and visualize SQL queries without leaving the desktop, streamlining database exploration and debugging. Ideal for developers and data analysts who prefer a simple, cross‑platform GUI over command‑line tools.
