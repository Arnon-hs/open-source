## 📝 Summary

SensibleJS is an open-source project that enables reactive user interface (UI) development using plain HTML attributes, resulting in a lightweight solution that fits within a 10KB footprint. Built to simplify frontend development, this library leverages the power of HTML attributes to create dynamic and interactive UI components. By utilizing SensibleJS, developers can focus on writing clean, declarative code without the need for complex frameworks or libraries.
