## 📝 Summary

Nova is an open-source server-side modding framework developed for Paper servers. It simplifies the process of adding custom content such as items, blocks, and GUIs by leveraging resource pack tricks, eliminating the need for client-side modifications. This framework aims to streamline the development process for modders and server administrators alike.
