## 📝 Summary

Mangiucugna/json_repair is a lightweight Python library that automatically detects and fixes malformed JSON generated by large language models, APIs, logs, or user input. It provides simple functions to sanitize, correct common syntax errors, and return valid JSON objects, making data handling more robust in AI/ML pipelines, backend services, and development tools. The package is open‑source, easy to integrate, and designed to streamline error‑prone JSON processing.
