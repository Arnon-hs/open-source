## 📝 Summary

CRC (Container Runtime Configuration) is an open-source tool designed to simplify container management. It uses local VMs to run OpenShift 4.x clusters, Microshift, or Podman environments, ideal for testing and development purposes. This tool enables users to efficiently set up and manage containerized environments for AI/ML and DevTools projects.
