## 📝 Summary

PhoneClaw is an open‑source, on‑device AI assistant for smartphones that runs locally using the lightweight Gemma‑4 model. It provides a responsive, privacy‑focused conversational agent capable of handling typical phone tasks—such as messaging, scheduling, and information retrieval—without needing cloud services. The project is built with a focus on efficiency, enabling real‑time AI interactions on resource‑constrained mobile hardware.
