## 📝 Summary

GenOrca/unreal-mcp is an open‑source MCP (Message Control Protocol) server that lets you drive Unreal Engine 5 projects through Claude and other AI agents. It provides a flexible API for both Python and C++ developers, enabling the creation of custom AI‑powered tools, automation scripts, and backend services that interact directly with UE5.
