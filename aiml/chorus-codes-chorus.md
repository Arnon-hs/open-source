## 📝 Summary

Chorus is an open‑source DevTools utility that enables multi‑LLM peer review of code changes. By integrating with any command‑line interface you prefer, Chorus automatically summons 2–4 additional language models to critique, suggest improvements, and validate your code before it’s merged or shipped. This collaborative AI workflow helps catch bugs and enforce best practices, streamlining the development process.
