## 📝 Summary

CodeAuto is an open‑source intelligent programming assistant built in Java, inspired by Claude Code and Minicode. It leverages AI/ML techniques to understand developer intent and generate, refactor, or debug code across multiple languages, offering a seamless, IDE‑integrated experience. Designed as a dev‑tool with a lightweight frontend, CodeAuto aims to boost productivity for developers of all skill levels.
