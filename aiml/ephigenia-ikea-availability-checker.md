## 📝 Summary

Ephigenia/ikea-availability-checker is a command‑line tool and reusable library that lets users query IKEA’s inventory to see whether specific products are in stock at chosen stores or across countries. It provides a simple API for developers to integrate real‑time availability checks into scripts, automation pipelines, or other applications. The project is open‑source and geared toward backend and DevOps workflows.
