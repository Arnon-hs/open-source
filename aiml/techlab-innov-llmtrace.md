## 📝 Summary

techlab-innov/llmtrace is an open‑source, zero‑code proxy that adds security and observability layers to any OpenAI‑compatible LLM API. Written in Rust, it monitors traffic in real time to detect prompt injection attacks, scan for personally identifiable information, and enforce cost‑control policies, all without requiring changes to existing client code. The project fits both backend and frontend use‑cases within AI/ML applications.
