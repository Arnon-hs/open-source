## 📝 Summary

Arkon is an open‑source platform that lets organizations centrally govern how employees interact with any AI client. Through a single web portal, admins can define resource limits, access policies, and workspace contexts, while employees connect once via the Model Context Protocol (MCP) to receive the appropriate AI context automatically. It streamlines AI usage across teams, improving security, compliance, and productivity.
