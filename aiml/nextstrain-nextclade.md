## 📝 Summary

Nextclade is an open-source project that utilizes AI/ML for viral genome analysis. It provides a suite of tools for aligning viral genomes, identifying mutations, assigning clades, conducting quality checks, and placing them on phylogenetic trees. This comprehensive framework enables researchers to trace the evolution and spread of viral strains.
