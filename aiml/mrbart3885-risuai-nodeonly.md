## 📝 Summary

Risuai‑NodeOnly is an open‑source, user‑friendly desktop application that enables you to create and run custom story‑driven role‑play sessions with large language models. Forked from the original Risuai project, it focuses on a lightweight Node‑only implementation, providing developers with easy‑to‑configure AI/ML tools and a clean interface for designing interactive narratives without the overhead of additional back‑end services.
