## 📝 Summary

Locus is an open‑source Unity development agent that leverages AI/ML techniques to automate routine coding tasks, generate scripts, and assist with scene setup directly within the Unity editor. By integrating natural‑language prompts and contextual code analysis, it helps developers accelerate prototyping and streamline workflow without leaving the Unity environment.
