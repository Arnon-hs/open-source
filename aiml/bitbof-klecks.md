## 📝 Summary

**Klecks** is an open‑source, community‑funded digital painting application that powers the popular web‑based art platform Kleki.com. Built with modern web technologies, it offers a responsive canvas, a rich set of brushes, layers, and real‑time AI‑enhanced features such as smart color blending and style transfer. The project invites contributors to improve the tool’s performance, extend its artistic capabilities, and keep the painting experience free and accessible to everyone.
