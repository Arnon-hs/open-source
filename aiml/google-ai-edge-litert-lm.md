## 📝 Summary

LiteRT‑LM is an open‑source library that brings lightweight, on‑device inference capabilities to large language models using Google’s AI Edge tooling. It provides a streamlined runtime and optimization pipeline that enables developers to run transformer‑based models efficiently on constrained hardware such as smartphones, microcontrollers, and edge AI accelerators. By integrating quantization, kernel fusion, and hardware‑specific backends, LiteRT‑LM makes it practical to deploy sophisticated language‑model functionality directly at the edge.
