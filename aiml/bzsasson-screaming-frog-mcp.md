## 📝 Summary

Screaming‑Frog‑MCP is an open‑source MCP (Message Communication Protocol) server that extends the Screaming Frog SEO Spider with programmatic control. It lets you initiate crawls, retrieve detailed crawl data, and manage stored crawl results through Claude or any MCP‑compatible client, making large‑scale SEO automation and integration into backend workflows straightforward.
