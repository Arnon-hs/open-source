## 📝 Summary

Tenere is an open‑source terminal user interface (TUI) designed to interact with large language models (LLMs) directly from the command line. It provides a lightweight, keyboard‑driven environment for prompting, exploring, and managing LLM responses, making AI/ML workflows more accessible to developers and power users. Built with modern frontend tools, Tenere integrates seamlessly into DevOps pipelines and other developer tooling.
