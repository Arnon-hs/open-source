## 📝 Summary

vfarcic/dot‑ai is an open‑source, AI‑powered agent that streamlines application deployments across any Kubernetes cluster. By combining dynamic cluster discovery with plain‑English governance policies, it lets developers and operators describe desired state in natural language while the agent translates those intents into concrete Kubernetes manifests and actions. The tool bridges AI/ML and DevOps, providing a unified, dual‑mode interface for consistent, policy‑driven deployments in heterogeneous environments.
