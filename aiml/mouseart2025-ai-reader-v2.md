## 📝 Summary

**AI‑Reader‑V2** is an open‑source visualization suite for analyzing Chinese AI‑generated novels, offering interactive character‑relationship graphs, geographic maps, timelines, and an encyclopedia‑style knowledge base. Built with a React frontend, FastAPI backend, and SQLite storage, it seamlessly integrates both local Ollama models and the ten most popular cloud LLM providers. The project provides a ready‑to‑use platform for developers and researchers interested in narrative analytics and AI‑driven storytelling tools.
