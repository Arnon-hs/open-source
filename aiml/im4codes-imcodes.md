## 📝 Summary

Imcodes is an open‑source framework that enables agents to communicate and collaborate through a shared memory store, allowing supervised execution of tasks and transparent cross‑agent auditing across multiple AI providers. It simplifies building multi‑agent systems by handling state synchronization, execution control, and provenance tracking, making it easier to develop, debug, and trust complex AI workflows.
