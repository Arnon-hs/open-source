## 📝 Summary

RedSOC is an open-source project that successfully demonstrates prompt injection attacks on AI System-on-Chip (SoC) assistants, achieving a 100% success rate. This vulnerability affects AI-powered assistants integrated into various systems, compromising their security and potentially leading to unauthorized access or data manipulation. The project aims to raise awareness about the security risks associated with AI SoC assistants.
