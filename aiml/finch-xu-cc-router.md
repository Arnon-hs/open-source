## 📝 Summary

finch‑xu/cc‑router is an open‑source smart routing service that runs locally and aggregates the token, coding, and API quotas of multiple LLM providers—including DeepSeek, Qwen, Kimi, MiMo, MiniMax, GLM, and Claude—into a single virtual plan. It lets users mix and match different model families (e.g., Opus, Sonnet, Haiku) across three slots, scheduling requests sequentially or via round‑robin, while handling rate‑limiting and automatic failover to fully utilize every available quota. This tool simplifies multi‑provider AI usage for backend applications.
