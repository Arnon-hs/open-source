## 📝 Summary

StyleSwift is an open‑source browser extension that lets users instantly apply a new, attractive skin to any website using a single natural‑language prompt. It combines AI‑driven smart parsing with precise CSS generation while keeping user data private, offering a fully transparent, privacy‑first alternative to tools like Stylish.
