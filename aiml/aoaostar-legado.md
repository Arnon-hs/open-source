## 📝 Summary

**Legado** is an open‑source reading application focused on aggregating and managing book sources (书源) for Chinese literature. It provides a modular framework that lets users add, customize, and switch between multiple content providers, enabling a seamless reading experience across various ebook platforms. The project leverages AI/ML techniques for source discovery and content recommendation, making it a versatile tool for developers and avid readers alike.
