Here's a brief summary for the open-source project:

SkillGuard is an open-source tool designed to scan agent skills for potential prompt injection payloads, helping to mitigate security risks associated with malicious input. This project aims to identify vulnerabilities in AI-powered chatbots and voice assistants, ensuring the integrity of user interactions. By utilizing SkillGuard, developers can enhance the security of their applications and protect users from potential attacks.
