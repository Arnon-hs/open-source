## 📝 Summary

Inspektor Gadget is an open-source framework for collecting data and inspecting systems on Kubernetes clusters and Linux hosts. Utilizing eBPF (Extended Berkeley Packet Filter), it provides a set of tools for in-depth monitoring and analysis. This platform enables developers to gain valuable insights into cluster and system performance.
