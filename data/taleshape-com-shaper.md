## 📝 Summary

The shaper project by taleshape-com is an open-source tool that enables users to visualize and share data directly from SQL queries. Built on top of the DuckDB database, it streamlines the process of data exploration and sharing. With shaper, users can easily transform and display data without having to switch between different tools or interfaces.
