## 📝 Summary

Zeugma440/atldotnet is a fully managed, portable C# library that enables developers to read, edit, and manipulate audio data and metadata (tags) across a wide range of audio formats, playlists, and CUE sheets. Designed for ease of use, it abstracts the complexities of handling different file types while providing a consistent API for common audio‑processing tasks. The library is open‑source, making it suitable for integration into .NET applications that require robust audio metadata management.
