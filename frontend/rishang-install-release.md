## 📝 Summary

Rishang/install-release is a lightweight command‑line utility that streamlines the installation and updating of single‑binary CLI tools hosted on GitHub or GitLab. It works across Linux, macOS, and Windows Subsystem for Linux, automatically fetching the latest release assets, handling version checks, and placing executables in the user’s preferred directory. Ideal for developers who need a simple, cross‑platform way to manage standalone command‑line tools without manual downloads.
