## 📝 Summary

The open-source project is a RISC-V emulator capable of running the classic first-person shooter game DOOM. This emulator allows developers to experience the iconic game on a low-level, processor-agnostic platform, enabling exploration of the RISC-V instruction set architecture and its capabilities. By emulating DOOM on RISC-V, the project showcases the versatility and potential of this emerging processor architecture.
