## 📝 Summary

The OpenApoc project is an open-source rebuild of the XCOM Apocalypse game engine, allowing developers to modify and extend the original code. This rebuild requires the original game files to run, providing a unique opportunity for modders and enthusiasts to experiment with the game's engine. By open-sourcing the project, developers can collaborate and contribute to the engine's improvement.
