## 📝 Summary

Grummage is an open-source, interactive terminal frontend for Grype, a popular vulnerability scanner. This project aims to provide a user-friendly interface for scanning and analyzing vulnerabilities, making it easier to manage security risks. By leveraging Grype's scanning capabilities, Grummage offers a streamlined experience for identifying and addressing potential security threats.
