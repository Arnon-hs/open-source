## 📝 Summary

OtakuWorld is an open-source project that offers a unified user interface for three distinct media consumption applications: an anime watcher, a manga reader, and a novel reader. This mobile project allows users to enjoy their favorite anime, manga, and novels across various platforms, providing a seamless and consistent experience. By leveraging the same UI across multiple apps, users can easily navigate and find content without needing to adjust to new interfaces.
