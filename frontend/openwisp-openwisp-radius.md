## 📝 Summary

Here's a 2-3 sentence summary for the openwisp/openwisp-radius project:

OpenWISP RADIUS is an open-source project providing a comprehensive administration web interface and REST API for FreeRADIUS 3, built on top of Django and Python. It supports various authentication methods, including captive portal, WPA Enterprise, social login, and Hotspot 2.0, making it a versatile solution for network administrators. The project also enables user registration, importing users from CSV files, and integrates with FreeRADIUS's rlm_rest module for extended functionality.
