**Summary:**  
*Meridian* is an open‑source tool that lets you deploy a fully configured, censorship‑resistant proxy server using the VLESS + Reality protocol with a single command. It works out‑of‑the‑box for both direct and relay node setups, making it a convenient backend solution for developers who need robust, privacy‑focused networking.
