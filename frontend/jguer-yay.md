## 📝 Summary

Yet another Yogurt, or "yay" for short, is an open-source project that provides a user-friendly command-line interface for managing packages on Arch Linux's Arch User Repository (AUR). Written in the Go programming language, "yay" offers a convenient alternative for users seeking to automate package installation and updates. With its simple and intuitive design, "yay" streamlines the process of managing AUR packages.
