## 📝 Summary

jjui is an open-source, text-based user interface (TUI) designed for interacting with the Jujutsu version control system. This project provides a user-friendly and command-line interface alternative for managing and navigating Jujutsu repositories. By utilizing a TUI, users can efficiently interact with Jujutsu without the need for a graphical interface.
