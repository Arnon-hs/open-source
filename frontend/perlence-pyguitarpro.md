## 📝 Summary

Perlence/PyGuitarPro is an open-source project that enables users to read, write, and manipulate Guitar Pro files, including GP3, GP4, and GP5 formats. This project provides a convenient tool for musicians and developers to work with Guitar Pro files, making it easier to create, edit, and share music scores. With its frontend capabilities, users can easily interact with the file manipulation functionalities.
