## 📝 Summary

ComfyUI‑Trellis2 is an open‑source wrapper that integrates the Trellis 2 framework into the ComfyUI environment, providing a seamless frontend interface for building and visualizing workflows. It streamlines the use of Trellis 2’s capabilities within ComfyUI, making it easier for developers and designers to create, edit, and run complex UI pipelines without leaving the ComfyUI ecosystem.
