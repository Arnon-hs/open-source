## 📝 Summary

GeorgeEnglezos/Scrcpy‑GUI is an open‑source, beginner‑friendly graphical front‑end for the popular Scrcpy screen‑mirroring tool. It provides a simple, cross‑platform UI that lets users connect, control, and configure Android devices without using the command line, making mobile debugging and streaming more accessible. The project focuses on clean frontend design while leveraging Scrcpy’s powerful backend features.
