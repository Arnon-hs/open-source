## 📝 Summary

ImGuizmo is an open-source, immediate mode 3D gizmo library built on top of the Dear ImGui framework. It provides a user-friendly interface for scene editing and other controls, making it ideal for game development, 3D modeling, and other visualization applications. ImGuizmo offers a flexible and customizable solution for adding interactive 3D manipulation tools to your project.
