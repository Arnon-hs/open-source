## 📝 Summary

yigger/jiezhang is an open‑source WeChat Mini Program built with the Taro framework that provides a clean, lightweight personal accounting solution. It offers core bookkeeping features—such as income/expense tracking, categorization, and simple data visualization—while leveraging Taro’s cross‑platform capabilities for a smooth frontend experience. The project is geared toward developers looking for a ready‑made, Taro‑based template for financial record‑keeping in the WeChat ecosystem.
