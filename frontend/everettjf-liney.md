## 📝 Summary

Liney is a customizable macOS terminal workspace designed for developers who work with multiple repositories, allowing for seamless worktree switching and a distraction-free coding experience. Built using AppKit, SwiftUI, and Ghostty, Liney offers a native and intuitive interface for users to manage their terminal workflow. This open-source project aims to enhance productivity and focus for developers working on complex projects.
