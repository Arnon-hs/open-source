## 📝 Summary

Integrate is a lightweight, pure‑Rust crate that provides simple, high‑performance numerical integration utilities. It offers a clean API for approximating definite integrals using common methods such as the trapezoidal rule and Simpson’s rule, making it ideal for scientific and engineering calculations without pulling in large dependencies. The library is easy to integrate into frontend Rust projects where fast, reliable integration is needed.
