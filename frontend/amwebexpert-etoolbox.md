## 📝 Summary

amtoolbox is an open‑source toolbox of web‑development utilities built with React, Ant Design, and TypeScript. It leverages TanStack Query for data fetching and Zustand for state management, offering reusable components and helper functions that streamline frontend, dev‑tool, and mobile UI workflows. The library is designed to be lightweight, highly configurable, and ready for integration into modern React projects.
