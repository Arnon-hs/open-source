## 📝 Summary

Twine is an open‑source, multiplatform RSS reader crafted with Kotlin and Jetpack Compose, offering a sleek, native UI across Android, desktop, and web. It aggregates and displays feeds efficiently, leveraging Compose’s declarative framework for smooth, responsive interactions while keeping the codebase shared across platforms.
