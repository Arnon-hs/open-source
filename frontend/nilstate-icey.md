## 📝 Summary

Icey is an open-source, real-time media stack designed as a lightweight alternative to libwebrtc. Built in C++20, it provides a robust and efficient solution for frontend applications requiring real-time media processing. With its modular architecture, Icey enables seamless communication between users, making it an ideal choice for real-time communication platforms.
