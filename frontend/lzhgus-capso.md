## 📝 Summary

Here's a brief summary for the Capso project:

Capso is an open-source, native screenshot and screen recording tool for macOS, offering a free alternative to CleanShot X. Built with Swift 6.0 and SwiftUI, Capso provides a user-friendly interface and seamless functionality. This project aims to provide a reliable and customizable solution for capturing and recording screenshots on macOS.
