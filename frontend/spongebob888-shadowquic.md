## 📝 Summary

spongebob888/shadowquic is an open‑source proxy that leverages the QUIC transport protocol to provide ultra‑low latency (0‑RTT) connections while disguising traffic using SNI camouflage. It enables seamless forwarding of TCP or UDP streams through a QUIC tunnel, making it useful for bypassing censorship or obfuscating network flows without requiring complex configuration. The project is positioned as a lightweight frontend tool for fast, stealthy communication.
