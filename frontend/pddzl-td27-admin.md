## 📝 Summary

Here's a brief summary of the project in English:

This is a decentralized dynamic routing and permission management system, built using a combination of GoLang, Gin, Casbin, Vue3, TypeScript, Element Plus, and Pinia. The system utilizes a front-end and back-end separation architecture to provide a robust and scalable solution for managing dynamic routes and user permissions. The project is open-source and available for developers to use and contribute to.
