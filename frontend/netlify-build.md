## 📝 Summary

Netlify Build is a Node‑based engine that executes your site’s build command, runs Build Plugins, and bundles Netlify Functions into production‑ready artifacts. It can be invoked locally via the Netlify CLI or automatically in Netlify’s Buildbot, streamlining both frontend and backend workflows for modern web projects.
