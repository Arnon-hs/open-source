## 📝 Summary

jpochyla/psst is a lightweight, cross‑platform Spotify client built with a native graphical user interface, offering fast performance and low resource usage. It provides all essential playback controls, playlist management, and search capabilities while running on Linux, macOS, and Windows without requiring a web browser. The project is open‑source, making it a handy tool for developers and power users who prefer a minimalist, desktop‑first Spotify experience.
