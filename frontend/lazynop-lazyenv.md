## 📝 Summary

The lazyenv project is an open-source, text-based user interface (TUI) designed to simplify the management of .env files. This utility allows users to easily create, edit, and manage environment variables in a .env file through a command-line interface. By streamlining this process, lazyenv aims to save developers time and reduce errors when working with environmental settings.
