## 📝 Summary

ComfyUI‑UniRig is an open‑source wrapper that integrates the UniRig rigging system directly into the ComfyUI visual programming environment. It provides a user‑friendly frontend for creating, editing, and managing rig configurations without leaving the ComfyUI workflow, streamlining the rigging process for artists and developers.
