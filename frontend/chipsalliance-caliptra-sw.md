## 📝 Summary

The Caliptra-sw project is an open-source initiative that provides software components for building and testing Caliptra hardware. It includes the ROM, FMC (Fabric Management Controller), and runtime firmware, as well as libraries and tools necessary for development. This project aims to facilitate collaboration and customization of Caliptra software.
