## 📝 Summary

jasonish/evebox is a web‑based graphical interface for visualizing Suricata EVE JSON events stored in Elasticsearch. It provides searchable, filterable tables, timeline views, and customizable dashboards to help security analysts investigate network alerts and incidents efficiently. The project focuses on a responsive frontend that simplifies threat hunting and log analysis for Suricata users.
