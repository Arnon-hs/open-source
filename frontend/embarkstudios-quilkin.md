## 📝 Summary

Quilkin is an open-source UDP proxy designed to enhance security, access control, and data collection for large-scale multiplayer game server deployments. This non-transparent proxy provides critical features such as telemetry data, metrics, and more, ensuring a robust and controlled gaming environment. Developed by Embark Studios, Quilkin is a valuable tool for game developers looking to optimize and secure their dedicated game servers.
