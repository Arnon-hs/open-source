## 📝 Summary

This open-source project, justi/claude-code-project-boundary, is a plugin for the Claude Code editor that provides scope-aware protection. It allows users to perform destructive operations like deleting, moving, and changing file permissions within a project, while blocking these actions outside the project scope. This plugin is designed for use in the dangerouslySkipPermissions mode.
