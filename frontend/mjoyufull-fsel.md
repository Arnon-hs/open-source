## 📝 Summary

Mjoyufull/fsel is a lightweight, fast terminal‑user‑interface (TUI) application launcher designed for GNU/Linux and *BSD systems. It provides a responsive, keyboard‑driven interface for quickly searching, previewing, and launching installed programs, making it a handy front‑end tool for developers and power users. The project is open‑source and focuses on minimal dependencies and high performance.
