## 📝 Summary

dotmake‑build/command-line provides a declarative way to build System.CommandLine interfaces using attributes, turning classes into commands and their properties into options or arguments without any runtime reflection. A built‑in source generator automatically converts these annotated types into strongly‑typed CLI code, delivering fast, compile‑time safety and minimal boilerplate. Ideal for developers who need a clean, front‑end‑focused toolchain for creating robust command‑line utilities.
