## 📝 Summary

Moor is an open‑source platform for creating shared, programmable online spaces, offering a web‑based front‑end that is compatible with the classic LambdaMOO virtual world system. It lets users build and interact within persistent, customizable environments using familiar MOO scripting, while providing modern web technologies for seamless access. Development has now moved to Codeberg at https://codeberg.org/timbran/moor.
