## 📝 Summary

Turbo is an experimental text editor that combines the Scintilla editing component with the classic Turbo Vision UI framework. It provides a lightweight, console‑style interface while leveraging Scintilla’s powerful syntax highlighting and editing features. Ideal for developers who want a retro‑styled, highly customizable frontend for code editing.
