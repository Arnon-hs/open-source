## 📝 Summary

Hayabusa is an open-source threat hunting and fast forensics timeline generator designed for Windows event logs. Based on the Sigma framework, it enables rapid analysis and visualization of potential security threats by generating a comprehensive timeline of system events. This tool empowers security professionals to efficiently identify and respond to potential incidents.
