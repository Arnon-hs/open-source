## 📝 Summary

nielsfaber/alarmo is an open‑source integration that adds a simple, customizable alarm system to Home Assistant, allowing users to arm, disarm, and monitor security states directly from the Home Assistant UI or automations. It provides straightforward configuration, supports multiple alarm modes, and integrates seamlessly with existing Home Assistant entities and notifications. This makes setting up a reliable, home‑grown security solution fast and accessible for both beginners and advanced users.
