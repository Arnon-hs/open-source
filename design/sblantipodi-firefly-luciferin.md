## 📝 Summary

sblantipodi’s **firefly_luciferin** is a high‑performance Java application that captures the PC’s screen in real time and streams the color data to the Glow Worm Luciferin firmware. Paired with the firmware, it drives LED strips to provide precise bias lighting and immersive ambient lighting that matches on‑screen content. The project is open‑source and geared toward users looking to build a low‑latency, customizable lighting system for their computer setup.
