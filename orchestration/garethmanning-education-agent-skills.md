## 📝 Summary

GarethManning/education-agent-skills is an open‑source library that provides a curated collection of evidence‑based Claude AI skills specifically tailored for educators. The skills are designed to help teachers automate routine tasks, generate instructional content, and support personalized learning, while also enabling seamless orchestration of multiple agents in larger educational workflows. The project combines AI/ML techniques with thoughtful instructional design to create practical, teacher‑friendly automation tools.
