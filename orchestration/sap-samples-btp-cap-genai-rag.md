## 📝 Summary

This open-source project, SAP-samples/btp-cap-genai-rag, offers a collection of GenAI samples on the SAP Business Technology Platform (SAP BTP). It demonstrates the integration of Large Language Models (LLMs) via SAP AI Core and LangChain in SAP Cloud Application Programming Model (CAP), as well as advanced techniques like Retrieval Augmented Generation (RAG) for single and multitenant environments. The project provides a valuable resource for developers to explore and build upon GenAI capabilities on SAP BTP.
