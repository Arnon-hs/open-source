## 📝 Summary

zhikuncode is an open‑source, self‑hosted multi‑agent AI programming assistant that serves as a drop‑in replacement for Claude Code. It offers both CLI and web UI access (even from a mobile browser), supports any LLM, native browser automation, over 90 built‑in commands, extensible skills/plugins, persistent memory, and enterprise‑grade security, all deployable with a single Docker command.
