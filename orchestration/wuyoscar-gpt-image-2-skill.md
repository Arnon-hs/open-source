## 📝 Summary

wuyoscar/gpt_image_2_skill is an open‑source toolkit that lets developers generate, edit, and organize image prompts for OpenAI’s image models. It provides a searchable prompt gallery, reusable “skill” modules for common image‑generation tasks, and a command‑line interface to orchestrate these capabilities within larger AI workflows.
