sickn33/antigravity-awesome-skills is an open‑source, installable library that provides more than 1,400 ready‑to‑use “agentic” skills for a range of AI coding assistants—including Claude Code, Cursor, Codex CLI, Gemini CLI, Antigravity, and others.
