## 📝 Summary

SpectaQL is an open‑source tool that automatically generates clean, static documentation for GraphQL APIs directly from schema files and query examples. By processing GraphQL introspection data and markdown annotations, it produces ready‑to‑host, searchable HTML pages that keep API docs in sync with code changes. Ideal for backend teams looking to streamline documentation workflows without manual upkeep.
