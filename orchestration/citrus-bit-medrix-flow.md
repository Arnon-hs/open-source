## 📝 Summary

MedrixFlow is a full‑stack AI‑agent orchestration platform that leverages LangGraph on the backend to coordinate multiple agents, manage state, and provide persistent memory across conversations. Its modern frontend, built with Next.js 16, offers an interactive UI where agents can run sandboxed code, browse the web, and manipulate files within isolated thread‑level environments. The system is designed for seamless, context‑aware AI workflows across both front‑ and back‑end components.
