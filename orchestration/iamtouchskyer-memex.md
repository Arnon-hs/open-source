## 📝 Summary

i​amtouchskyer/memex is an open‑source, Zettelkasten‑style knowledge store that provides persistent, markdown‑based memory for AI coding agents such as Claude Code, Cursor, VS Code Copilot, Codex, Windsurf, and any MCP client. It avoids heavyweight vector databases by syncing plain markdown files through Git, making the system lightweight, version‑controlled, and easy to integrate into existing development workflows. The project falls under orchestration, AI/ML, and backend tooling.
