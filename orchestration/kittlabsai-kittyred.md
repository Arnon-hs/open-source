## 📝 Summary

KittLabsAI/KittyRed is an open‑source, local‑first platform that lets users host and manage AI pets and companion agents in a unified workspace. It integrates Claude Code, Codex and GitHub Copilot, providing a multi‑agent orchestration layer for development, automation, and interactive assistance. Designed for developers and hobbyists, KittyRed combines AI/ML, orchestration, and dev‑tool capabilities into a single, privacy‑focused environment.
