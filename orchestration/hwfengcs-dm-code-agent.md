## 📝 Summary

hwfengcs/DM‑Code‑Agent is a lightweight (~1,500 lines) Python code‑generation agent that combines ReAct prompting, a planning module, Reflexion for self‑correction, and a hybrid Retrieval‑Augmented Generation (RAG) approach. It includes built‑in support for SWE‑bench Lite evaluation and trace replay, making the system both auditable and easy to benchmark. The project targets orchestration of AI‑driven coding workflows and can be used as a frontend component for interactive code assistance.
