## 📝 Summary

A lightweight open‑source tool that scrapes any public website to capture its design tokens—colors, typography, spacing, border‑radius, and shadows—and outputs them as JSON and CSS custom properties for easy reuse in local projects. It can be run as a standalone CLI or invoked as an AI‑agent skill for Claude, Cursor, Codex, and similar platforms, streamlining design‑system extraction in orchestration, AI/ML, and frontend workflows.
