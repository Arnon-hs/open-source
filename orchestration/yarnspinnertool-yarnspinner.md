## 📝 Summary

Yarn Spinner is an open-source project that provides the core compiler and engine-agnostic components for the Yarn Spinner dialogue tool. It allows developers to create and manage interactive dialogue systems for their projects, enabling more engaging and immersive experiences. This tool is designed to support a wide range of applications and engines.
