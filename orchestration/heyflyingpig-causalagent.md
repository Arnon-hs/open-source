## 📝 Summary

Here's a brief summary for the Heyflyingpig/CausalAgent project:

CausalAgent is an open-source causal analysis framework powered by the LangGraph protocol. It leverages multiple tools, including MCP and RAG, to assist in causal analysis, and provides users with a comprehensive causal analysis report and visual graph. This framework enables users to navigate and understand complex causal relationships in a structured and interpretable manner.
