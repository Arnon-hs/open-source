## 📝 Summary

Here's a brief summary for the open-source project:

Omni-skills-forge is an open-source, universal skill and slash-command manager designed for AI coding assistants. It enables users to build, share, and install community skills, making it a powerful tool for developers. Compatible with Claude Code, omni-skills-forge simplifies the process of creating and integrating AI-powered tools into coding workflows.
