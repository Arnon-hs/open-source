## 📝 Summary

openwong2kim/wmux is a lightweight, Windows‑native alternative to tmux designed for AI‑driven development workflows. It lets users split the console into multiple panes to run Claude Code, Codex, Gemini CLI, and an MCP‑based browser automation side‑by‑side—without needing WSL. The tool focuses on orchestrating AI agents and their outputs in a unified, front‑end terminal UI.
