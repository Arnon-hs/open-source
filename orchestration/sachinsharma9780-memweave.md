## 📝 Summary

Memweave is an async‑first, zero‑infrastructure Python library that equips AI agents with persistent, searchable memory by storing data as plain Markdown files. It lets developers add long‑term context to agents without setting up databases or external services, and provides simple APIs for adding, retrieving, and querying the stored notes. Ideal for orchestration, AI/ML workflows, and DevOps pipelines where lightweight, file‑based knowledge bases are preferred.
