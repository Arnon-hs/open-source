## 📝 Summary

jmerelnyc/Photo‑agents is an open‑source framework for autonomous, self‑evolving AI agents that integrate vision‑grounded perception with layered memory and dynamically generated skills. By leveraging large language models, the system can write its own tool‑use functions and orchestrate complex workflows directly on a user’s computer, enabling adaptive, multimodal assistance for a wide range of tasks.
