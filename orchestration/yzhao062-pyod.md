## 📝 Summary

pyod is an open‑source Python library that provides a comprehensive suite of more than 60 anomaly‑detection algorithms, covering tabular, time‑series, graph, text, and image data. It includes a benchmark‑validated ADEngine for orchestrating detectors and an agentic workflow that enables AI agents to select, combine, and deploy models automatically. The project targets developers who need a versatile, plug‑and‑play solution for building and scaling anomaly‑detection pipelines.
