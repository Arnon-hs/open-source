## 📝 Summary

The vscode-mcp project is an open-source, MCP server designed to enhance the coding experience for AI coding agents, such as Claude Code, VSCode, Cursor, and Windsurf. It enables real-time Language Server Protocol (LSP) diagnostics, type information, and code navigation, eliminating the need for slow tsc/eslint checks. This project facilitates seamless integration with editor self-functionality, streamlining AI coding workflows.
