## 📝 Summary

A lightweight JavaScript library for currency conversion, **cashify** provides a simple, dependency‑free API to format, parse, and convert monetary values across different currencies. It builds on the concepts of the original *money.js* but adds modern features such as support for custom exchange‑rate sources, locale‑aware formatting, and TypeScript typings. Ideal for trading and fintech applications, cashify lets developers handle currency calculations accurately and with minimal overhead.
