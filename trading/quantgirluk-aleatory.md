## 📝 Summary

**quantgirluk/aleatory** is an open‑source Python library that provides tools for simulating and visualising stochastic processes such as random walks, Brownian motion, and other probabilistic models. Designed for traders, analysts, and data scientists, it offers a simple API for generating synthetic time‑series data, applying statistical transformations, and creating interactive plots to explore the behaviour of random systems. The package is lightweight, well‑documented, and integrates smoothly with common data‑science stacks (NumPy, pandas, Matplotlib/Plotly).
