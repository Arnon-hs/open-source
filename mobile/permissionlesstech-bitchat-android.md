## 📝 Summary

**permissionlesstech/bitchat-android** is an open‑source Android app that creates a peer‑to‑peer chat network using Bluetooth Mesh, allowing users to communicate without any central server or internet connection. Inspired by the simplicity of IRC, it provides real‑time, channel‑style messaging that works locally across multiple devices, making it ideal for offline, ad‑hoc group conversations.
