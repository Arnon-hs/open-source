## 📝 Summary

CSMWrap is an open-source project that restores the functionality of the PC BIOS on UEFI-only systems. It provides a solution for users who need to access legacy BIOS features, making it easier to work with older hardware on newer devices. This project caters to those in the mobile sector who require such functionality.
