## 📝 Summary

SukkaW/Surge provides a collection of ready‑to‑use rule snippets and full rule sets compatible with popular proxy tools such as Surge, Mihomo (Clash.Meta), Clash Premium, sing‑box, Surfboard for Android, and Stash. Designed for mobile users, the repository offers organized, regularly updated configurations that simplify traffic routing, domain filtering, and ad‑blocking across these platforms. It serves as an open‑source resource for anyone looking to quickly implement efficient network rules on their Android or iOS devices.
