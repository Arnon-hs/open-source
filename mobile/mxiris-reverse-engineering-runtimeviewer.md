## 📝 Summary

MxIris‑Reverse‑Engineering/RuntimeViewer is an open‑source utility for macOS and iOS that lets developers inspect the Objective‑C and Swift runtimes of a running application. It provides a graphical interface to explore classes, methods, properties, and protocols in real time, facilitating debugging, reverse‑engineering, and learning of runtime behavior. The tool is lightweight, integrates with Xcode, and supports both iOS simulators and physical devices.
