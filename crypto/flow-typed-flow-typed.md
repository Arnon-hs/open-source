## 📝 Summary

flow-typed is an open‑source repository that provides high‑quality, community‑maintained type definitions for JavaScript libraries used with the Flow static type checker. By centralizing these definitions, it enables developers to add accurate type information to third‑party packages, improving code safety and tooling support across projects. The project serves as a key DevTools resource for managing and sharing Flow type definitions.
