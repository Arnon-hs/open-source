## 📝 Summary

**wayfinder.nvim** is a Neovim plugin that streamlines code navigation by providing on‑the‑fly access to definitions, references, callers, related tests, and Git information for the symbol under the cursor, while maintaining persistent "Trails" that let you backtrack through your exploration history. It integrates seamlessly with Neovim’s LSP ecosystem, offering a lightweight, configurable UI for developers who need fast, context‑aware insight while browsing large codebases.
