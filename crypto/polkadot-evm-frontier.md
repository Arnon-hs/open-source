## 📝 Summary

polkadot-evm/frontier is an open‑source compatibility layer that brings Ethereum’s EVM functionality to the Polkadot and Substrate ecosystems. It enables developers to run existing Ethereum smart contracts and tooling natively on Polkadot parachains, facilitating cross‑chain interoperability and broader DeFi integration.
