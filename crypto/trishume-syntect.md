## 📝 Summary

Syntect is an open-source Rust library that leverages Sublime Text syntax definitions to provide syntax highlighting capabilities. This library allows developers to easily integrate robust syntax highlighting into their applications, utilizing the extensive and well-curated syntax definitions provided by Sublime Text. By utilizing the existing Sublime Text syntax repositories, Syntect reduces the need for developers to create and maintain custom syntax highlighting rules.
