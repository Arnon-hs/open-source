## 📝 Summary

keep‑starknet‑strange / starknet‑agentic is an open‑source toolkit that equips AI agents with the ability to interact directly with the Starknet blockchain. It bundles smart‑contract libraries, deployment utilities, and reusable skill modules so autonomous agents can execute, monitor, and manage transactions on Starknet without manual intervention. The project bridges crypto and AI/ML, enabling developers to build intelligent, on‑chain agents for decentralized finance, NFTs, and other Web3 use cases.
