## 📝 Summary

The **truss44/mcp-crypto-price** project is an open‑source Model Context Protocol (MCP) server that leverages CoinCap’s API to deliver real‑time cryptocurrency data. It allows Claude and any MCP‑compatible client to request current prices, analyze market trends, and retrieve historical price information for seamless crypto‑trading and analytics integration.
