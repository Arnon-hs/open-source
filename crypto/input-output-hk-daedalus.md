## 📝 Summary

Daedalus is an open-source cryptocurrency wallet specifically designed for ADA, a native cryptocurrency of the Cardano blockchain. Built to be community-driven and user-friendly, Daedalus enables secure storage, sending, and receiving of ADA, as well as access to various blockchain features. This wallet serves as a core tool for ADA holders to interact with the Cardano ecosystem.
