## 📝 Summary

loderunner/scrt is a lightweight, command‑line secret manager designed for developers, system administrators, and DevOps teams. It provides a simple yet secure way to store, retrieve, and manage encrypted credentials directly from the terminal, integrating smoothly into CI/CD pipelines and other automation workflows. Built with modern cryptographic primitives, scrt focuses on ease‑of‑use without compromising security.
