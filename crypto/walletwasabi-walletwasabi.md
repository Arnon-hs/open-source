## 📝 Summary

WalletWasabi is an open‑source, non‑custodial Bitcoin wallet that focuses on enhancing user privacy through advanced coin‑joining techniques such as Chaumian CoinJoin. Available for Windows, Linux, and macOS, it offers a user‑friendly interface, robust security features, and transparent, community‑driven development. The project aims to make private, secure Bitcoin transactions accessible to everyday users.
