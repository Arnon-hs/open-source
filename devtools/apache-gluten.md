## 📝 Summary

Gluten is an open‑source middleware that bridges JVM‑based SQL engines (such as Spark or Flink) with high‑performance native execution back‑ends. By translating the logical and physical plans generated by the SQL engine into native code, it offloads compute‑intensive workloads to optimized native runtimes, delivering faster query execution while preserving the familiar JVM ecosystem.
