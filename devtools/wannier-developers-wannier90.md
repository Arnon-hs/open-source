## 📝 Summary

The Wannier90 project is an open-source code for calculating maximally localized Wannier functions, a powerful tool in solid-state physics for understanding electronic structure and band topology. This official repository provides a comprehensive set of tools for researchers to analyze and visualize electronic properties of materials, facilitating breakthroughs in fields like materials science and condensed matter physics. By leveraging Wannier90, users can extract essential information from first-principles simulations and experimentally determined electronic structures.
