## 📝 Summary

The ansys/pyaedt project is an open-source Python client package that enables users to interface with the ANSYS AIM and HFSS software through the ANSYS Workbench environment. This package provides a Python API for users to automate and customize their simulations, data analysis, and workflows. By leveraging ansys/pyaedt, users can streamline their design and analysis processes, improving efficiency and productivity.
