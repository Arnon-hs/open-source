## 📝 Summary

SteamInputDB is an open-source project that simplifies the process of discovering, filtering, and applying community-created SteamInput controller configurations. The project includes a web database (steaminputdb.com) and a companion app, SteamInputDB-Buddy, which integrates directly with the Steam client for ease of use. This resource enables users to share and apply custom controller layouts with a seamless experience.
