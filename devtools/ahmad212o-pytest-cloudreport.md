## 📝 Summary

Pytest-cloudreport is an open-source project that extends the capabilities of pytest, a popular Python testing framework. It enables the generation of local HTML reports and provides flaky-test detection features, allowing developers to identify and address unstable tests. This tool aims to improve the testing experience and efficiency for AI/ML and DevTools projects.
