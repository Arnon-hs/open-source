## 📝 Summary

RepoBar is an open‑source utility that places real‑time GitHub repository information—such as CI build status, open issues, pull requests, and the latest release—directly in your menu bar and terminal prompt. By integrating these metrics into everyday workflows, developers can monitor project health at a glance without leaving their working environment. It’s a lightweight DevTools addition for macOS, Linux, and Windows terminals.
