## 📝 Summary

This open-source project, eslint-functional/eslint-plugin-functional, provides ESLint rules that prioritize functional programming (fp) in JavaScript and TypeScript. By disabling mutation and promoting fp principles, developers can write more predictable, scalable, and maintainable code. The plugin aims to make functional programming a natural part of the development workflow.
