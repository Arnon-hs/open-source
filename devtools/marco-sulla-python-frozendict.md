## 📝 Summary

Marco‑Sulla/python‑frozendict provides a lightweight, drop‑in replacement for Python’s built‑in dict that is immutable (hashable) and can be safely used as keys in other dictionaries or elements of sets. It mirrors the standard dict API for reading data while preventing any modifications, making it ideal for configuration objects, caching, and functional programming patterns. The project is open‑source, well‑documented, and packaged as a pure‑Python DevTools library.
