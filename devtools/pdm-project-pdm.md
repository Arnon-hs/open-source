### English

pdm-project/pdm is an open-source, modern Python package and dependency manager that supports the latest PEP standards. This tool streamlines the process of managing Python dependencies, making it easier for developers to install, update, and manage packages in their projects. With its support for PEP 518 and PEP 517, pdm provides a reliable and efficient way to manage dependencies for Python projects.
