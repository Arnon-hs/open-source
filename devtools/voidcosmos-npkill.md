## 📝 Summary

The npkill project is a free, open-source tool designed to help developers identify and manage large node_modules directories in their system. It lists all node_modules directories and their sizes, allowing users to select and delete unnecessary packages to free up disk space. This tool aids in maintaining a clutter-free and organized development environment.
