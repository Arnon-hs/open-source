## 📝 Summary

GlassHaven (Haven) is an open‑source Android application that provides a unified, free client for connecting to remote machines via SSH, VNC, RDP, and SFTP. Designed for developers and power users, it consolidates multiple protocols into a single, lightweight interface, making it easy to manage servers, desktops, and file transfers on the go. The project is actively maintained and licensed for community contributions.
