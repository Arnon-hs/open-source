## 📝 Summary

Project_LemonLime is an open-source, lightweight judging system designed for competitive programming, particularly for OI (Online Judge) competitions. Built upon Lemon and LemonPlus, it supports three major desktop systems and aims to provide an efficient evaluation experience. This system is intended for developers and competitors alike, facilitating a streamlined judging process for online programming challenges.
