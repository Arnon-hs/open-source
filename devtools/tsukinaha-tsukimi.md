## 📝 Summary

Tsukimi (tsukinaha/tsukimi) is a lightweight, third‑party client for Jellyfin designed specifically for Linux users. It provides a clean, native interface to browse, stream, and manage media libraries without the overhead of a full web browser. The project is open‑source and aims to integrate smoothly with typical Linux desktop environments.
