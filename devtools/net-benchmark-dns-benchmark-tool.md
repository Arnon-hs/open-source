## 📝 Summary

The **net‑benchmark/dns‑benchmark‑tool** is a high‑speed, open‑source utility for measuring DNS performance across a wide range of query types, including DNSSEC validation and modern encrypted protocols such as DoH and DoT. It provides detailed latency and reliability metrics, supports bulk testing of multiple resolvers, and includes enterprise‑grade features like configurable test suites, result export, and integration hooks for CI/CD pipelines. Ideal for developers, DevOps engineers, and infrastructure teams who need comprehensive, repeatable DNS benchmarking and security verification.
