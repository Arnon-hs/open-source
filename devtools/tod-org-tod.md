## 📝 Summary

tod‑org/tod is an open‑source, unofficial command‑line client for Todoist written in Rust. It lets users manage tasks, projects, and labels directly from the terminal with fast, type‑safe commands and full Todoist API support. Ideal for developers and power users who prefer a lightweight, scriptable interface for their Todoist workflow.
