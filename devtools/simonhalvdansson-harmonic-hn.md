## 📝 Summary

Harmonic‑HN is a modern Android client for browsing Hacker News, offering a clean, Material‑Design interface and smooth navigation optimized for mobile devices. Built with Kotlin and Jetpack Compose, the open‑source app provides features such as offline caching, comment threading, dark mode, and customizable sorting, making it a handy tool for developers and news enthusiasts on the go.
