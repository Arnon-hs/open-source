## 📝 Summary

EmbarkStudios /cargo-deny is a Rust Cargo plugin that scans a project's dependency graph for known security vulnerabilities, license incompatibilities, and other compliance issues. By integrating directly into the Cargo workflow, it helps developers enforce safe and policy‑compliant dependencies early in the build process. It’s especially useful for DevTools and mobile Rust projects that require rigorous dependency auditing.
