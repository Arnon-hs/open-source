## 📝 Summary

The Atostek/ros2-client is an open-source ROS2 client library built on top of RustDDS, providing a Rust-based interface for interacting with ROS2 systems. This library enables developers to create robust and efficient ROS2 clients, facilitating seamless communication between Rust applications and ROS2-based robot operating systems. By utilizing RustDDS, the library ensures reliable, real-time data exchange and synchronization.
