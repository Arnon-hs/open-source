## 📝 Summary

Terraform-docs is an open-source project that automatically generates documentation from Terraform modules, simplifying the process of maintaining accurate and up-to-date documentation for infrastructure as code. It supports various output formats, allowing users to easily customize and share their documentation across different platforms. This tool is ideal for DevOps teams and infrastructure engineers seeking to streamline their documentation workflow.
